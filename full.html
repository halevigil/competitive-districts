Model spec

All partisanship / ideology quantities are in percentage points. Districts: m values drawn from N(μdistrict, σdistrict²) form the right half (clipped to [0, 100]); the left half is the negation, with dm = 0. Sum is exactly 0 by symmetry.

Candidates (log form, % units):
cDi = +βDswing log(100/|di|) + βDcompet log(100/|di+v|) + ε,  ε~N(μD,σD)
cRi = −βRswing log(100/|di|) − βRcompet log(100/|di+v|) + ε,  ε~N(μR,σR)

log(100/|d|) is large (~6.9) when |d|→0 (very competitive district) and ≈0 when |d|→100 (very safe district). |d| is clipped to ≥0.1% for numerical stability. With β>0 for both parties (the defaults), candidates moderate in competitive districts. The model is D↔R symmetric under d→−d, r→−r when μR = −μD, σR = σD, βR = βD, v = 0.

Note: This is a pure unit change from the previous [-1,1] formulation — the simulation behavior is identical because both numerator and denominator inside the sigmoid scale by 100.

Winner: P(R) = σ((v + di − wmod(cD+cR)) / σnoise)

Median rep r̃ = ideology of rep at sorted-by-ideology rank m.
